## To run application locally
### Requirements:
- Node.js
- PostgreSQL
### Instructions:
- open terminal
  - 'createdb errant'
  - 'npm install'
  - 'npm run seed'
  - 'npm run start:dev' (if Win, open 2nd terminal 'nodemon server/index.js')
- open browser and navigate to 'http://localhost:3000'
